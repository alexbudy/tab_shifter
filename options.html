Move Tab Between Windows

Enable Ctrl+Shift+Space - bring current tab to new window and back
Enable Ctrl+Shift+[Alt]+Down - move current tab to next Chrome Window [use Alt to keep focus on previous window]

Move Tab in Window

Enable Ctrl+Shift+Right/Left - move active tab all the way to the right or left

Move Window

Enable Ctrl+Shift+< or > - move current Chrome window left/right between monitors
Enable Ctrl+Shift+' or / - maximize/shrink by % the current Chrome window
